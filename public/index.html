WundergroundGmaps

Click the Map to Select a Location

This web app combines two technologies, the Google Maps API and the Wunderground API. When you the client click the map, the map returns coordinates. These coordinates are sent to the server which then passes them in query string and makes a request to Wunderground for the weather based on the coordinates given. The server then send back the weather in the form of JSON back to you the client. Then the JSON is parsed and put into a string that you can read.
